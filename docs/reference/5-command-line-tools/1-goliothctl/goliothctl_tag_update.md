## goliothctl tag update

Update the name of an existing tag.

```
goliothctl tag update [tag name] [flags]
```

### Examples

```
> goliothctl tag update my-tag --name formally-known-as-my-tag
```

### Options

```
  -h, --help          help for update
      --name string   new tag name
```
